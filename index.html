ПОСЛЕДНИЕ ИЗМЕНЕНИЯ

v1.14
Новые возможности:
- Новый менеждер плагинов. Можно скрывать ненужные плагины.
- Все плагины идут вместе с клиентом (благодаря новому менеджеру).
- Результат копирования в буфер обмена из окна клиента попадает в 
системную переменную $BUFFER, чтобы использовать в триггерах.
- Начальные игровые профили упакованы в архив для компактности.
- SDK отдельным архивом идет вместе с клиентом.
- Обновлена справка клиента.
- Плагин автопереноса (autowrap.lua) можно настраивать на блок склейки на ключевым строчкам.

ОСНОВНЫЕ ВОЗМОЖНОСТИ КЛИЕНТА

  • Быстрый и легкий. Не будет "тормозить" даже на слабых компьютерах.
  • Современный. Понимает кодировк UTF-8 и WIN, умеет использовать протоколы MCCP, MSDP, MTTS.
  • Встроен скриптовый движок LUA
  • Тут есть триггеры(actions), макросы(alises), замены(subs), антизамены(antisubs), фильтры(gags), подсветки(highlights), таймеры(timers), автоподстановки(tabs), группы(groups).
  • Триггеры могут быть написаны в обычном стиле или регулярным выражением.
  • Поддержка триггеров(в том числе многострочных) с обработкой на LUA.
  • Поддержка плагинов, можно написать любой плагин для более комфортной игры.
  • Уже доступны плагины звука, синтезатора голосом, панели кнопок для игры мышью, полоски со здоровьем, лор, фильтры команд и многое другое. Уже больше 20 плагинов.
  • Система профилей позволяет хранить настройки для разных мадов.
  • Запись логов игры в html и txt
  • История мада. Можно читать историю совместно с игрой.
  • История команд, с автоподстановкой из истории.
  • Многофункциональная командная строка.
  • Есть справка к клиенту.
  • Есть SDK для написания плагинов и документация к нему.
  • В SDK есть IDE Decoda адаптированная для написания плагинов на Lua для клиента.
