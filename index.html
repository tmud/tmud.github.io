ОСНОВНЫЕ ВОЗМОЖНОСТИ КЛИЕНТА

  • Быстрый и легкий. Не будет "тормозить" даже на слабых компьютерах.
  • Современный. Понимает кодировку UTF-8, умеет использовать протоколы MCCP, MSDP, MTTS.
  • Тут есть триггеры, макросы, замены, антизамены, фильтры, подсветки, таймеры, автоподстановки, группы.
  • Триггеры могут быть написаны в обычном стиле или регулярным выражением.
  • Поддержка триггеров(в том числе многострочных) с обработкой на Lua.
  • Поддержка плагинов, можно написать любой плагин для более комфортной игры.
  • Уже доступны плагины звука, синтезатора голосом, панели кнопок для игры мышью, полоски со здоровьем, лор, фильтры команд и др. Уже больше 20 плагинов.
  • Система профилей позволяет хранить настройки для разных мадов.

ПОСЛЕДНИЕ ИЗМЕНЕНИЯ

v1.02
Исправлены ошибки:
- Исправлена неправильная отрисовка игровой строки в клиенте, если в строке встречался символ
амперсанда (&).
- Исправлена ошибка в работе lua-триггеров, которая могла вызвать пропажу строк и многократное 
срабатывание самого триггера.
Новые возможности:
- Изменилось окно, которое появляется при закрытии настроек, если не был сохранен текущий триггер,
или если изменился шаблон строки триггера. Теперь можно сохранить/не сохранять новый триггер или 
отменить закрытие окна настроек клиента.

v1.0
- Поддерживаются основные элементы для настройки игры:
  - Триггеры (actions)
  - Замены (subs) с поддержкой цвета
  - Подсветки (highlights), включая подчеркивание, курсив, обрамление
  - Фильтры (gags)
  - Антизамены (antisubs)
  - Макросы (aliases)
  - Таймеры (timers)
  - Автоподстановки (tabs)
  - Группы (groups)
- Кодировки сервера - win и utf8.
- Протоколы MCCP, MSDP, MTTS, OSC palette.
- Работа с двойным IAC (буква я в win кодмровке)
- 256 цветов с палитрой первых 16 цветов.
- Плавающие output-окна.
- Запись логов игры в форматах html и txt.
- Работа с буфером обмена. Вставка из буфера и копирование в буфер обмена нужной части текста.
- История мада. Можно читать историю совместно с игрой.
- История команд, с автоподстановкой из истории.
- Многофункциональная командная строка.
- Система профилей.
- Плагины (более 20 штук):
  - автоперенос длинных строк
  - работа со звуком (звуковые эффекты, фоновая музыка)
  - синтезатор речи по тексту (говорилка)
  - игровая панель для игры мышкой
  - дополнительная панель для нужной инфы, с поддержкой тикера
  - панель с полосками здоровья, энергии и другого
  - лор база вещей
  - маршруты (speedwalks)
  - и другие, есть даже игра Сапер.
- Есть справка к клиенту.
- Есть SDK для написания плагинов и документация.
- В SDK есть IDE Decoda адаптированная для написания плагинов на Lua для клиента.
